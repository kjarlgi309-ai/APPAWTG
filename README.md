이게 뭔가요

아이 둘 키우는 아빠가 직접 만들었습니다. 금요일 밤마다 "아이와 가볼만한 곳"을 검색하다가 광고 글 사이에서 길을 잃는 게 지겨워서요.


🗺️ 전국 1,000곳 이상 — 직접 가본 곳 + 공공데이터 기반 큐레이션
🔍 3초 필터 — 지역 · 실내/실외 · 아이 나이 · 예산만 고르면 끝
⛅ 주말 날씨 연동 — 토/일 기온·강수확률 자동 표시, 비 오면 "실내만 보기" 추천
🎲 랜덤 추천 — 결정에 어려움이 있는 부모를 위한 "그냥 정해줘" 버튼
